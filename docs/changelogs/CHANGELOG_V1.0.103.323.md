# Changelog v1.0.103.323

## Calendário
- Unificamos scroll vertical/horizontal no grid de calendário para preservar cabeçalhos e coluna fixa em uma única área de rolagem.
- Ajustamos offsets/z-index dos stickies para evitar invasão do menu lateral e manter as três faixas superiores visíveis em qualquer rolagem.
- Removemos as cores temporárias de debug das três linhas superiores (barra principal, datas e regras em lote), voltando ao tema neutro.

## Notas
- Sem mudanças de API ou de schema.
- Testado no modo calendário com rolagem vertical e horizontal.
